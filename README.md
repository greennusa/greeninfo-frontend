## Development
- git clone
- npm install
- create `servers.json` file in /public with format
```json
{
  "servers": [
    {
      "name": "server1",
      "url": "http://localhost:3000"
    }
  ]
}
```
- npm run dev

## Deployment
- git clone
- npm install
- create `servers.json` file in /public with format
```json
{
  "servers": [
    {
      "name": "server1",
      "url": "http://localhost:3000"
    }
  ]
}
```
- npm run build
- copy files inside /out to your server
